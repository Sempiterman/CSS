Lista de comandos de Git

Con estos comandos configuramos o cargamos los valores inicales del entorno

git init

git config --global user.name"Nombre"

git config --global user.email Correoelectronico@mail.com

Con el siguiente git config --list checamos que tengamos realmente la configuracion lista para trabajar una vez hecha continuar

Con este comando le estamos diciendo que archivo estamos incluyendo dando su nombre y extension del mismo para empezar el control de versionamiento sobre el nombre del archivo dado

git add nombre.extension

Con este comando asignas el nombre o comentario asignado al archivo previamente adicionado con add recuerda siempre tener un distintivo relacionado con el archivo que te recuerde el cambio

con el siguientes comando consultas el estado de los directorios

git status -s

para agregar un cambio en la version primero se debe teclear

git add nombre.extension

despues

git commit -m "Comentario o nombre que te ayude a identificar el cambio sobre la version"

cada que quieras cargar un cambio

Con el siguiente comando solicitas las peticiones versionadas o en su defecto te imprime el listado con el nombre de cambio dado con el comentario que se le dio

git log --oneline

con este solamente indicas las version a cargar OJO: tomar en cuenta que si cargas una anterior desara las posteriores
